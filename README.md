# Portfolio Website

This is a simple personal portfolio website created using **HTML** and **CSS**. The website is designed to present basic information about me and provide a way for visitors to contact me.

The project focuses on practicing fundamental front-end development skills such as structuring web pages with HTML and styling them using CSS.

## Pages

**Home Page**

* Displays my introduction and basic information.
* Highlights the purpose of the portfolio.

**Contact Page**

* Provides contact details or a simple contact form for visitors to reach me.

## Technologies Used

* HTML5
* CSS3

## Purpose

The main goal of this project is to improve my understanding of basic web development and build a simple portfolio website as a starting point for future projects.
